## HOW TO CONTRIBUTE ##

1. CLONE REPOSITORY
2. INSTALL PREQUISITES
3. INSTALL DEPENDENCIES
4. CONFIGURE .ENV
5. TEST IF IT RUNS LOCALLY
6. CREATE BRANCH FOR THE CHANGE THAT YOU WILL MAKE (ALWAYS BRANCH THE DEVELOP BRANCH)
7. WORK ONLY ON THAT BRANCH
9. WHEN DONE, TEST IF IT WORKS LOCALLY
8. WHEN THE CHANGES ARE DONE, PUSH YOUR BRANCH
9. CREATE PULL REQUEST
10. AFTER CODE REVIEW, YOUR CHANGES WILL BE MADE IN THE MAIN REPO AN BRANCH
